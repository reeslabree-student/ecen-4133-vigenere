# vigenere.py
``vigenere.py`` is a python script to determine the key for a vigenere string using Kasiski's test to determine key length, and letter frequency analysis for determining the key.

## Usage
```
python3 vigenere.py

<paste in vigenere text>
ctrl+c
```
